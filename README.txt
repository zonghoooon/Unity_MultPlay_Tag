This is a multiplayer tag game made using the Photon server.😎

Once every 10 seconds, a bomb is created on a randomly selected player.💣
A player with a bomb can hand over the bomb by touching another player.🖐

The time remaining until the bomb explodes is displayed in the upper right corner.⏰
At the top center, the current number of players is displayed. 👨🏻‍🤝‍👨🏻

WASD: Move 🧎‍♂️
Left Shift: Sprint 🏃‍♂️
